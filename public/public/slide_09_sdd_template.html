Spec (What do you want?)

Create a React component that displays project cards.
Data comes from JSON.
Each card shows: image, title, description, links.
Responsive design (mobile + desktop).
2

Acceptance Criteria

  • Cards are displayed correctly in a grid layout
  • 1 row on mobile, 3 rows on desktop
  • Clicking on a card opens the project detail view
  • No red errors in the console
3

Task Breakdown

  • Create the component file structure
  • Add CSS grid layout and styling
  • Create a mock data JSON file
  • Make it responsive (media queries)
  • Test with different screen sizes
4
