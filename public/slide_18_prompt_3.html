PROMPT 3: ADMIN AUTH & CRUD

prompt_admin.txt
Build the ClearContent CMS admin panel (e.g. /admin or /cms) with Supabase Auth. Requirements: 1. Login: Auth (e.g. Supabase); roles: superadmin, admin, editor. 2. Visual builder: Add, remove, reorder blocks; layouts, spacing, colors, typography. 3. Content: Edit all texts, buttons, offer list, footer; menu editor (labels, order, links). 4. Form constructor: Contact form fields (add/remove/reorder), required, consent, success message. 5. SEO & multilingual: Page title, meta description; enable languages, edit per language. 6. Audit: last updated (and who, if feasible).

Functionality

Secure login
Block and content management
Forms for data editing

Acceptance Criteria

  • Login works (role-based access)
  • Visual builder: add/reorder blocks
  • Form constructor and SEO per language
  • Public site reflects CMS after refresh
